# airship-dev-tools

Set of internal tools for Airship and metal3 development

## Setup

```
$ ./tools/init-repo.sh
```

## Update the nordix master branches

```
$ ./tools/update-nordix-repos-master.sh
```

## Run a development container

```
$ ./container/run-workspace.sh
```
